# 第二街区
import { Callout } from 'nextra-theme-docs'
import { Tab, Tabs } from 'nextra-theme-docs'

第二街区是一个以多势力组合在一起为核心的城市区域，人们将这个区域称呼为联合区，这里集中了全都灵所有最具实力的都灵非官方势力

每一个势力在这里都有自己的管辖区域，在区域内势力的权利甚至大于政府的法律法规，但至少在都灵城府彻底倒台之前，明面上他们都是会选择遵守都灵法律，为维持都灵秩序出一份力的，不过它也因此成为了各路豪杰和冒险家们展示实力和谈判交易的重要场所。

## 教会

![1681496048303](../../public/image/命运无常异能世界规则书/1681496048303.png)

> 有形之物的毁灭，必然带来无形之物的消逝。众生忘却的，我们为彼等记忆；众生铭记的，我们为彼等保存。

以存续为核心教义的教派，在大崩溃后整个世界陷入“失去一切”的理论下，以让旧世界的一切于精神世界中重现的观念，使教义在人们精神崩溃的时刻成为了强大的精神支柱。

### 日冕教

日冕教在大崩溃初期展现出了惊人的社会稳定性与向心性，故很快成为了多数都市复兴的核心力量，其教义突出的精神世界观也使其难以受到物质基础的影响，政府势力也倾向于稳定与拉拢教会势力。<br></br>
同时，在其虔诚的信众与日俱增的情况下，渐渐成为了泛意识海中小小的涟漪，对旧世界的渴望最终形成了一个巨大的异常集团，在这种渴望下，人类以比预想的快不少的速度完成了新世纪的重建。但同时，与旧世界不同的部分的发现也因此变得缓慢，当日冕教不再可控时，终究会重蹈大崩溃的覆辙。

### 圣城【圣士坦德堡】

最高教皇驻地，作为最高精神领袖，负责教义的制定与制度的修改。

### 新都灵【都灵教区】

位于都灵的大主教区，负责都灵新城的教会管理与教义传播。除第一环区外，所有环区皆有教堂设立，是直接面向信众的机构，每个教堂有神父与修女管理日常事务，传道以及管理信众。

## 泰坦重工

主要重型大型设备的研发和生产，和军部签署了长期战略合作协议，是官方重要设施和武器军火的核心供给企业

## 燧王科技

复合型科技企业，主要业务是收集都灵旧址内的残留科技资料，复现或者研发符合需求的新科技产品。虽然进度极其缓慢，但是每次推出的核心产品都能引发一场科技地震。

## 博益医疗

主打科技医疗产品的企业，但是真正核心的产品是异常衍生物相关的医疗研发，包含但不限于生物组织再生/克隆器官/异常器官移植等一些违背人伦但非常好用的技术研发。

## 东风集团 （PL：闹钟）

![1681499222013](../../public/image/命运无常异能世界规则书/1681499222013.png)

东风集团由黄道先生和左季谷先生合伙创办，是致力于各式送货服务的大型企业集团，为都灵区块的客户们提供高品质高速度的服务。

> 关于子公司

包括但不限于东风外卖、东风特快、东风拉货、东风搬家，但只有东风外卖没有固定的办公地点，其他子公司在都灵的各个社区都有分部
子公司之一的东风特快是东风集团唯一主动招募异能者的公司。

## 都灵第一人民医院

都灵第一人民医院是一所位于都灵市第二街区的一家大型综合医院，是都灵规模最大、设备最先进的医院之一。医院拥有着高质量的医疗资源和丰富的医学经验，专业涵盖了各种科室，包括心脏病科、骨科、肿瘤科等等。此外，医院还拥有着优秀的医疗团队，为患者提供高效、安全的医疗服务。

## 教育系统

都灵最核心的教育系统汇集了全都灵最高端的教育资源，目前详细的教育系统缺乏补充，有意者可联系世界主参与共创。

### 都灵第一综合大学

都灵第一综合大学成立于1404年，是曾经欧洲历史最悠久的高等教育机构之一，同时也是都灵最好的大学，没有之一。它被认为是曾经意大利的“哈佛大学”。这所大学位于都灵市第二街区，拥有广阔的校园和先进的设施。它的研究生课程涵盖了众多领域，如人文、社会科学、医学、自然科学和技术等等。

### 都灵联合高中

这是一所非常著名的学校，吸引了很多有钱人和政府官员的子弟前来就读。这所学校位于意大利都灵市第二街区，是一所公立寄宿制高中。学校开设了多个专业领域的课程，包括自然科学、文化和艺术等方面。学生在这里可以接受到非常优质的教育资源和师资力量的支持，是目前都灵最受欢迎的高中之一。

### 都灵一中

一所设施齐全、教学质量高水平的中学，也是都灵所有富豪和官员的教育首选，这里为学生们提供了丰富的学习和发展机会。学校在精神、学术和体育方面都非常注重全面发展，提供广泛的学科课程和活动，以及一系列社区服务计划。

### 都灵二中

这是一所侧重于科学和技术的中学，为学生们提供了现代化的教学设施和实验室。该校强调实践和实验，为学生们提供了许多机会，让他们深入了解科学和技术领域。

### 都灵小学 

这所学校是为年龄在6到11岁之间的孩子们准备的，为他们提供了一种安全、友好和支持性的环境来学习和成长。该校的教师们非常注重孩子们的全面发展，提供了各种课程和活动，使孩子们在知识和技能方面都能够得到充分的发展。

### 都灵中央图书馆

<Callout type="info">
    由都灵利益联合体资助的大型图书馆，旨在重建都灵的过程中同时重建人类文明中的知识板块与普及教育，内容杂而不精，异能者与人类管理局负责协助管理<br></br>
所有公共图书馆资源共享，同时受都灵利益联合体管控，同一数据库储存所有藏书信息，不同的公共图书馆可根据读者需要，在读者付出运输费的情况下直接进行书籍的抽调<br></br>
所有公共图书馆外侧都有保安以及一小队的警署人员执勤，门口处有金属扫描安保系统，不得携带武器进入
